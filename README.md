## Topics covered so far
* SOLID Design Principles
* Factory Pattern
* Strategy Pattern

### Steps to run applications
```
Clone the Repo

Open project with your IDE

Make any necessary changes -> Save

Right click on the application Run as -> Spring Boot
```
