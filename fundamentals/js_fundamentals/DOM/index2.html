DOM (Document Object Model)

What is the DOM?

Treats an HTML, XHTML, XML document as a tree structure wherein each node is an object representing a part of the document.

  • Describes how all the HTML elements are related
  • Allows JavaScript to call into the DOM select elements and influence elements
  • DOM is created when the web page loads
  • Elements within the page can me accessed and changed

Node: Everything within the HTML is a node.
